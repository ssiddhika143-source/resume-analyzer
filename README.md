# ResumeAI — AI Powered Resume Analyzer

An AI-powered resume analyzer that matches your resume against job descriptions and provides detailed feedback.

## Features
- AI Match Score out of 100
- Matched and Missing Skills Detection
- Resume Red Flags Detector
- Interview Questions Generator
- ATS Optimization Tips
- Job Role Predictor
- Top Companies Matcher
- PDF Download of Results
- Dark/Light Mode

## Tech Stack
- Python, Flask, MySQL, Groq AI

## Setup
1. Clone the repo
2. Create virtual environment: python -m venv venv
3. Activate: venv\Scripts\activate
4. Install dependencies: pip install -r requirements.txt
5. Create config.py from config.example.py
6. Add your Groq API key
7. Set up MySQL database
8. Run: python app.py
